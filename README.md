# percival-osm

Componente OSM do **percival.OS** — servidor MCP para consulta do OpenStreetMap, buscando informações e pontos de interesse próximos a um local.

Baseado originalmente em [projectmoon's OSM tool for Open-WebUI](https://git.agnos.is/projectmoon/open-webui-filters/src/branch/master/osm.py).

## Funcionalidades

- Busca de pontos de interesse (POIs) por categoria (restaurantes, farmácias, hospitais, lojas, transporte, etc.)
- Geocodificação via Nominatim
- Rotas via OpenRouteService
- Interface MCP para integração com o nanobot/percival.OS
